# Dotfiles

## How to install

```bash
cd <path-to-dotfiles>
./install
```
### Tabnine setup

```
Tabnine::sem
```

Inside vim, type above to enable semantic completion(Language Server Protocol)

### Vimspector setup

```bash
cd ~/.vim/plugged/vimspector
./install_gadgets.py
```

Using command above, vimspector will be enabled(Debugger Adapter Protocol)
